# Bounty
A bounty is prize that can be won by anybody for completing a designated task.

## Fields

| Field | Description |
| ----- | ----------- |
| `bounty_id` | Unique id of the bounty. |
| `name` | The name of the bounty (e.g. "Ace Pot", "Turkey 9,10,11"). |
| `prize` | The reward for completing the bounty. |

## Relationships

None
